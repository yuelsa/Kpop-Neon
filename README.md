#  Kpop-Neon

# Kpop-Neon is a high-energy Android app that delivers fresh K-POP artist info and fandom data with a vibrant neon UI inspired by real light-stick colors.  
Designed with Jetpack Compose, the app emphasizes bold neon pink & neon blue visuals, modern UX, and smooth API integration.
---
# Features
- Search K-POP Artists &   
  Powered by the MusicBrainz API with K-pop filtering logic.
---
# Tech Stack
- Language: Kotlin  
- UI: Jetpack Compose  
- Network: Retrofit + Coroutines  
- Architecture: MVVM  
- API: MusicBrainz (free & public)
